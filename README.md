# Fin - Agente Financeiro Inteligente com IA Generativa

## Sobre o Projeto

Fin é um agente financeiro inteligente desenvolvido para ajudar usuários a entender melhor suas finanças, economizar dinheiro e tomar decisões mais conscientes sobre investimentos.

Utilizando IA Generativa, o agente analisa dados financeiros e oferece recomendações personalizadas de forma segura e objetiva.

## Principais Funcionalidades

- Análise de gastos mensais
- Identificação de padrões de consumo
- Sugestões de economia
- Recomendações de investimentos compatíveis com perfil
- Acompanhamento de metas financeiras
- Alertas proativos

## Tecnologias Utilizadas

- Python
- Streamlit
- Pandas
- JSON / CSV
- IA Generativa

## Estrutura do Projeto

- `data/` Base de dados mockada
- `docs/` Documentação do projeto
- `src/` Aplicação funcional
- `assets/` Recursos visuais
